# API Documentation

## Overview
This directory contains documentation for the web application.

## API Endpoints

### POST /api/form/submit
Submit form data

**Request Body:**
```json
{
  "name": "string",
  "email": "string"
}
```

**Response:**
```json
{
  "success": true,
  "message": "Form submitted successfully",
  "data": {}
}
```
